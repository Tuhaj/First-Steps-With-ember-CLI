What is ember CLI?

Tool for ember.js which origins from Ember App Kit (EAK) with:

  • strong conventional project structure
  • asset compilation based on broccoli

What can ember CLI give?

a new app in 4 steps!

Step I: install ember CLI

Step II: create a new app

Step III: after 2-3 minutes run the server

Step IV: Voilà!

Is it ok to use “ember generate” for beginners?

  • Some believe no, but yes! And pay attention ;-)
  • Learn from both sides: form particular to general and from the big picture to the details.

Expected gain in

  • Code organisation (folders and files tree)
  • Easier getting into project and understanding it
  • Code quality (ES 6 features)
  • Work flow (autosave, strong conventions, command line generators)

Some tips

  • Use original syntax, Coffee Script and Emblem will make your time worse
  • Embrace the conventions, you’re going to love them
  • Be careful with new versions…
